# 🐍 Snake Game

A modern responsive Snake Game built with HTML5 Canvas, CSS3, and Vanilla JavaScript.

This project was created as part of my frontend development journey while learning JavaScript and responsive web development. The game is optimized for both desktop and mobile devices with keyboard controls, touch controls, sound effects, score tracking, and a modern neon UI.

---

## 🚀 Live Features

* 🎮 Classic Snake gameplay
* 📱 Mobile-friendly touch controls (D-pad)
* ⌨️ Keyboard support
* 🔊 Dynamic sound effects using Web Audio API
* ⏸️ Pause / Resume system
* 🏆 Best score saved with LocalStorage
* ⚡ Multiple speed levels
* 🌙 Modern neon dark UI
* 📐 Responsive design
* 🎨 Smooth UI interactions and hover effects

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* HTML5 Canvas API
* Web Audio API
* LocalStorage API

## 🎯 Controls

### Desktop

| Key          | Action               |
| ------------ | -------------------- |
| ⬆️ ⬇️ ⬅️ ➡️  | Move Snake           |
| Spacebar     | Pause / Resume       |
| Start Button | Start / Restart Game |

### Mobile

* Use the on-screen D-pad buttons to control the snake.
